    Data Types

    Purpose / Description

    The EAV (Entity-Attribute-Value) system and 2sxc is all about data. Here some minimal documentation about internals.

    Difference Between Data-Type and Input-Type

    The data-type describes how data is stored (persisted) in various formats (SQL, JSON, XML) and how it's used in code (C#, JavaScript, Tokens, Angular, ...).

    The input-type describes how the user can enter such data - for example using a text-field or a date-picker. Learn more about the input types here.

    Basic Data Types

    EAV and 2sxc 10.0+ currently have the following data types

    1. Boolean
    2. Custom - for internal (core development) use only, please don't use for anything else
    3. DateTime - for dates and times
    4. Empty - for non-saved data like group-headings
    5. Entity - for relationships between items - like a book to the author or a blog-post to tags
    6. Hyperlink - a special string with helper objects which resolve "file:72" to the real link
    7. Number - for any kind of number like 1, 2, 3 or GPS coordinates
    8. String - for standard string types or when you other options don't work

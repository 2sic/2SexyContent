    Structured Content for .net Platforms

    Where to Start

    2sxc is an easy to use dynamic content system for .net Platforms like DNN. If you are just getting started, then this is probably not where you should be looking. Focus first on:

    1. Content on 2sxc.org like the learn section
    2. Apps on 2sxc Catalog which contains many ready-to-use apps like blogs and news, as well as tutorials
    3. The DNN Tutorials on 2sxc.org

    With that knowledge, you can already create amazing stuff.

    Everything you find here is very advanced and you won't need it for normal everyday work.

    Going Down the Rabbit Hole - C# APIs

    If you got past the basics and really want more, you should continue here:

    1. Read about the architecture or about internal data models
    2. Read about breaking changes
    3. Browse through the .net API
